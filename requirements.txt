# MicroPython Crash Detection System - Requirements

## Hardware Requirements

### Essential Components
- Raspberry Pi Pico 2 (RP2350-based) or Raspberry Pi Pico W
- MPU6050 6-axis Accelerometer/Gyroscope Module (I2C)
- Micro USB cable (for programming and power)
- Breadboard (for prototyping)
- Jumper wires (male-to-female recommended)

### Optional Components
- Active buzzer (3.3V, 5V with transistor)
- Battery pack (for portable operation)
  - 3x AAA batteries (4.5V) or
  - 2x AA batteries (3V) or
  - LiPo battery (3.7V) with voltage regulator
- External LED with 220Ω resistor (if not using built-in)
- Protective case/enclosure

## Software Requirements

### On Development Computer
- Python 3.7 or higher
- One of the following tools:
  - Thonny IDE 4.0+ (recommended for beginners)
  - mpremote (pip install mpremote)
  - rshell (pip install rshell)
  - ampy (pip install adafruit-ampy)

### On Raspberry Pi Pico 2
- MicroPython firmware v1.20.0 or later
  - Download from: https://micropython.org/download/RPI_PICO2/
  - File: rp2-pico2-latest.uf2

## Python Package Dependencies (for development tools)

```
# Install with: pip install -r requirements.txt (if using standard Python tools)
mpremote>=1.20.0
rshell>=0.0.31
```

Note: MicroPython on the Pico itself does not require external packages.
All necessary libraries (machine, time, math) are built into MicroPython.

## Pin Connections Summary

| Component | Pico Pin | GPIO | Notes |
|-----------|----------|------|-------|
| MPU6050 VCC | Pin 36 | 3.3V | Power supply |
| MPU6050 GND | Pin 38 | GND | Ground |
| MPU6050 SDA | Pin 1 | GP0 | I2C Data |
| MPU6050 SCL | Pin 2 | GP1 | I2C Clock |
| Buzzer (+) | Pin 20 | GP15 | Optional |
| Buzzer (-) | GND | GND | Optional |
| LED | Built-in | GP25 | Status indicator |

## I2C Address
- MPU6050: 0x68 (default) or 0x69 (if AD0 pin is high)

## Communication Settings
- I2C Frequency: 400 kHz (Fast Mode)
- UART Baud Rate: 115200 (for serial monitoring)

## Power Specifications
- Operating Voltage: 3.3V (regulated by Pico)
- Current Draw:
  - Pico 2: ~30mA (idle)
  - MPU6050: ~3.9mA (normal mode)
  - Total: ~35-40mA (excluding buzzer)
  - With buzzer active: +20-30mA

## Memory Requirements
- Flash: ~10KB for main.py
- RAM: ~5KB runtime memory
- Available on Pico 2: 2MB Flash, 264KB RAM (plenty of space)

## Environmental Specifications
- Operating Temperature: 0°C to 70°C (typical)
- MPU6050 Accelerometer Range: ±2g, ±4g, ±8g, ±16g (configurable)
- Project uses: ±4g range by default

## Optional Enhancements (Future)
- WiFi module (if using Pico W) for remote alerts
- GPS module for location tracking
- SD card module for data logging
- GSM module for SMS alerts
- OLED display for visual feedback
